# bye bye ai.

ai blocker by kiddokraft is a web extension that blocks ai related content.

download for firefox

[firefox add-on](https://addons.mozilla.org/firefox/addon/ai-blocker-by-kiddokraft/)


download for chrome

[if donations exceed 5 usd, kiddokraft can pay the $5 chrome developer fee to get the extension listed in chrome web store](https://liberapay.com/kiddokraft/donate)

download for safari

[if donations exceed 100 usd, kiddokraft can pay the $100 apple developer fee to get the extension listed in apple app store](https://liberapay.com/kiddokraft/donate)

### how it works

add any word to "follow the current trend and leave no fomo".


<img src="https://github.com/kiddokraft/ai-blocker/raw/main/demo/add-keyword.gif" width="200">


highlighting is the most satisfying way to see what is being blocked.

<img src="https://github.com/kiddokraft/ai-blocker/raw/main/demo/highlighting-mode.gif" width="200">

semantic blocking checks what is parallel in the dom tree and blocks the entire component.


<img src="https://github.com/kiddokraft/ai-blocker/raw/main/demo/semantic-blocking.gif" width="200">
